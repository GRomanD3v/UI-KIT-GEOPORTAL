# Proyecto de front-end con HTML y CSS Geoportal

## Descripción

Este proyecto es una página web que muestra una tarjeta con información sobre una alerta de seguridad y una grilla de íconos relacionados con el geoportal.

## Tecnologías utilizadas

- HTML
- CSS

## Estructura del proyecto

```
. 
├── index.html
├── styles.css
├── img-alerta.png
└── icons/
    ├── imagotipos-geoportal-horizontal_1-geoportal.png
    ├── imagotipos-geoportal-horizontal_2-geoprocesos.png
    ├── imagotipos-geoportal-horizontal_3-sinopsis-t.png
    ├── imagotipos-geoportal-horizontal_4-comando-c.png
    ├── imagotipos-geoportal-horizontal_5-clear-sky.png
    └── imagotipos-geoportal-horizontal_6-app-mobile.png
```

## Ejecución

1. Abrir el archivo `index.html` en un navegador web.
